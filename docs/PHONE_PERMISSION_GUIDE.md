# 微信小程序手机号获取权限申请指南

## 📋 申请条件

在申请之前，请确保你的小程序满足以下条件：

1. ✅ 小程序已通过微信认证（企业/个体工商户/政府/媒体/其他组织）
2. ✅ 小程序不属于个人主体（个人主体无法申请）
3. ✅ 有明确的使用场景需要获取用户手机号

---

## 🚀 申请步骤

### 第一步：登录微信公众平台

1. 打开浏览器，访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 使用小程序管理员账号登录
3. 进入小程序后台管理界面

### 第二步：进入接口设置页面

**路径**：
```
左侧菜单 → 开发 → 开发管理 → 接口设置
```

或者直接在地址栏输入：
```
https://mp.weixin.qq.com/wxopen/devprofile?action=get_profile
```

### 第三步：申请手机号接口

1. 在「接口设置」页面，找到 **"获取手机号"** 接口
2. 点击右侧的 **"去申请"** 或 **"申请权限"** 按钮

### 第四步：填写申请信息

需要填写以下信息：

#### 1. 使用场景描述（必填）
示例模板：
```
我们的小程序是一个烘焙电商平台，用户下单时需要获取手机号用于：
1. 订单配送时联系收件人
2. 发送订单状态变更通知
3. 会员积分变动提醒
4. 售后服务沟通

手机号是用户身份的唯一标识，也用于区分不同用户的会员权益和订单历史。
```

#### 2. 上传证明材料（选填，但建议上传）
可以上传的材料：
- 📄 营业执照复印件（带公章）
- 📄 小程序功能说明书
- 📄 隐私政策文档
- 📄 用户协议文档

#### 3. 勾选承诺
勾选 "本人承诺已阅读并遵守《微信小程序平台运营规范》"

### 第五步：提交审核

1. 点击 **"提交申请"** 按钮
2. 等待微信审核（通常 1-3 个工作日）
3. 审核结果会通过「站内信」和「管理员微信」通知

## 📱 替代方案（快速体验）

如果暂时无法申请到手机号权限，可以使用以下替代方案进行开发和测试：

### 方案 1：使用传统登录方式

修改登录页面，先使用 OpenID 登录：

```javascript
// pages/login/login.js
async onWechatLogin() {
  const result = await auth.doLogin();
  // 使用 OpenID 登录，不获取手机号
}
```

### 方案 2：手动输入手机号

在登录后添加手机号绑定页面：

```html
<!-- 手动输入手机号 -->
<view class="phone-input-box">
  <input
    type="number"
    placeholder="请输入手机号"
    maxlength="11"
    bindinput="onPhoneInput"
  />
  <button bindtap="sendSmsCode">获取验证码</button>
</view>
```

### 方案 3：使用微信测试号

1. 申请微信小程序测试号
2. 测试号有更多接口权限
3. 但仅用于开发测试，不能发布

申请地址：https://developers.weixin.qq.com/sandbox

## 🔍 审核被拒常见原因及解决

### 原因 1：使用场景描述不清晰

**拒绝提示**：
> "请详细描述使用场景，需说明获取手机号的必要性"

**解决方案**：
- 详细描述手机号的具体使用场景
- 说明为什么必须要手机号（如物流配送需要电话联系）
- 列举 3-5 个具体的使用场景

### 原因 2：小程序类目不符合

**拒绝提示**：
> "小程序类目不符合申请要求"

**解决方案**：
- 检查小程序类目是否选择正确
- 建议类目：电商、餐饮、生活服务、政务民生等
- 在「设置」→「基本设置」→「服务类目」中修改

### 原因 3：缺少隐私政策

**拒绝提示**：
> "请在小程序中展示隐私保护指引"

**解决方案**：
1. 创建隐私政策页面
2. 在登录页面添加隐私政策链接
3. 在小程序后台「设置」→「基本设置」→「用户隐私保护指引」中填写

## ⚡ 快速通过审核的技巧

1. **类目选择准确**
   - 选择与业务最接近的类目
   - 不要选择"工具"等模糊类目

2. **描述具体详细**
   - 不要使用模板化的描述
   - 结合自己的业务场景详细说明

3. **材料准备充分**
   - 上传营业执照增加可信度
   - 提供详细的功能说明文档

4. **功能已开发完成**
   - 申请时小程序应有基本功能
   - 不要是空壳小程序

## ❓ 常见问题

### Q: 个人主体小程序可以申请吗？

**A**: ❌ 不可以。只有企业、个体工商户、政府、媒体、其他组织主体可以申请。

### Q: 审核需要多长时间？

**A**: 通常 1-3 个工作日，高峰期可能需要更长时间。

### Q: 申请被拒后多久可以再次申请？

**A**: 可以立即修改后重新申请，没有冷却时间。

### Q: 一个账号可以绑定多个小程序吗？

**A**: 可以，每个小程序都需要单独申请权限。

### Q: 测试号需要申请吗？

**A**: 测试号自动拥有大部分接口权限，无需申请，但只能用于开发测试。
