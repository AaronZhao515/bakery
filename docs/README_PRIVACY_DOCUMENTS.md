## 🚀 使用步骤

### 第一步：部署隐私政策页面（5分钟）

1. 确保 `package-user/pages/privacy/` 下的文件已保存
2. 在微信开发者工具中预览小程序
3. 访问隐私政策页面，确认显示正常

### 第二步：填写用户隐私保护指引（10分钟）

1. 登录微信公众平台
2. 进入 `设置` → `基本设置` → `用户隐私保护指引`
3. 复制 `docs/PRIVACY_POLICY_CONCISE.md` 的内容
4. 粘贴到相应位置
5. 保存并提交

### 第三步：申请手机号权限（15分钟）

1. 进入 `开发` → `开发管理` → `接口设置`
2. 找到 `获取手机号` 点击 `去申请`
3. 复制下面的使用场景描述：

```
【小程序名称】暖心烘焙
【小程序类目】电商平台-食品

【使用场景详细描述】
我们的小程序是烘焙电商平台，用户下单购买面包蛋糕时需要获取手机号，主要用于：

1. 订单配送联系
   - 配送员需要联系收件人确认收货地址和配送时间
   - 确保商品能够准确、及时送达

2. 会员账户管理
   - 手机号作为用户唯一标识，区分不同用户
   - 关联会员积分、优惠券、订单历史
   - 提供个性化的会员服务

3. 订单状态通知
   - 订单发货、配送中、已送达等状态通知
   - 会员积分变动提醒
   - 优惠券到期提醒

4. 售后服务支持
   - 处理退换货需求时联系用户
   - 解答用户咨询和投诉

【获取必要性说明】
- 配送必须：烘焙食品需要及时配送，必须能够联系收件人
- 账户标识：手机号是最可靠的用户身份标识方式
- 服务合规：电商平台实名要求，需要真实联系方式

【数据保护措施】
- 数据加密存储于腾讯云数据库
- 仅用于上述明确目的
- 不与无关第三方共享
- 用户可随时申请删除账户
```

4. 上传营业执照（扫描件，建议加盖公章）
5. 勾选同意协议，提交申请

### 第四步：等待审核（1-3个工作日）

- 保持微信在线，接收审核通知
- 如被拒绝，根据反馈修改后重新提交

## ❓ 常见问题

### Q: 隐私政策页面显示空白？

A: 检查以下几点：
1. 文件是否已保存
2. 路径是否正确：`/package-user/pages/privacy/privacy`
3. 分包配置是否正确

### Q: 审核被拒绝了怎么办？

A: 查看拒绝原因，常见原因和解决方案：
1. "使用场景不清晰" → 详细描述手机号的具体使用场景
2. "缺少隐私保护指引" → 在后台填写用户隐私保护指引
3. "类目不符合" → 修改为电商平台或餐饮类目

### Q: 个人主体可以申请吗？

A: 不可以。只有企业/个体工商户/政府/媒体主体可以申请手机号权限。
