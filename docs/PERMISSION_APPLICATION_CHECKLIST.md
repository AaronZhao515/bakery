# 微信小程序手机号权限申请资料清单

## 📋 申请前准备

### 必备条件
- [ ] 小程序主体为企业/个体工商户/政府/媒体（个人主体不可申请）
- [ ] 完成微信认证
- [ ] 小程序已上线或有基本功能
- [ ] 已开通云开发或有后端服务

---

## 📁 申请资料清单

### 1. 使用场景描述（必填）

**填写位置：** 微信公众平台 → 开发 → 接口设置 → 获取手机号 → 申请

**模板内容：**
```
【小程序名称】暖心烘焙
【小程序类目】电商平台-食品

【使用场景详细描述】
我们的小程序是烘焙电商平台，用户下单购买面包蛋糕时需要获取手机号，主要用于：

1. 订单配送联系
   - 配送员需要联系收件人确认收货地址和配送时间
   - 确保商品能够准确、及时送达

2. 会员账户管理
   - 手机号作为用户唯一标识，区分不同用户
   - 关联会员积分、优惠券、订单历史
   - 提供个性化的会员服务

3. 订单状态通知
   - 订单发货、配送中、已送达等状态通知
   - 会员积分变动提醒
   - 优惠券到期提醒

4. 售后服务支持
   - 处理退换货需求时联系用户
   - 解答用户咨询和投诉

【获取必要性说明】
- 配送必须：烘焙食品需要及时配送，必须能够联系收件人
- 账户标识：手机号是最可靠的用户身份标识方式
- 服务合规：电商平台实名要求，需要真实联系方式

【数据保护措施】
- 数据加密存储于腾讯云数据库
- 仅用于上述明确目的
- 不与无关第三方共享
- 用户可随时申请删除账户
```

## 🚀 快速申请流程

### 第一步：准备资料（30分钟）
1. 复制上面的使用场景描述，根据实际情况修改
2. 确认隐私政策已上传到可访问的位置
3. 准备营业执照扫描件

### 第二步：填写申请（15分钟）
1. 登录微信公众平台
2. 进入「开发」→「开发管理」→「接口设置」
3. 找到「获取手机号」点击「去申请」
4. 粘贴使用场景描述
5. 上传营业执照（可选但建议）
6. 勾选同意协议
7. 提交申请

### 第三步：等待审核（1-3个工作日）
- 保持微信在线，接收审核通知
- 如被拒绝，根据反馈修改后重新提交

- 电商平台-食品

## 📞 常见问题解决

### Q1: 审核被拒"使用场景描述不清晰"

**解决：** 按照上面的模板重新填写，确保包含：
- 具体的使用场景（至少3个）
- 为什么必须用手机号
- 不用手机号会怎样

### Q2: 审核被拒"缺少隐私保护指引"

**解决：**
1. 在小程序内添加隐私政策页面
2. 在后台「设置」→「用户隐私保护指引」中填写
3. 确保隐私政策包含：收集信息、使用目的、保护措施、用户权利

### Q3: 审核被拒"类目不符合"

**解决：**
1. 进入「设置」→「基本设置」→「服务类目」
2. 添加或修改为：电商平台、餐饮服务、商家自营等
3. 确保类目与业务匹配

### Q4: 个人主体想申请

**解决：** 个人主体无法申请手机号权限，需要：
- 注册个体工商户
- 或使用企业主体重新注册小程序

## ⚠️ 注意事项

1. **申请前确保小程序有基本功能**，空壳小程序会被拒
2. **隐私政策必须真实有效**，不要随意复制网上的
3. **保持联系方式畅通**，审核人员可能会电话核实
4. **被拒绝后可以立即重新申请**，没有冷却时间
5. **审核期间不要修改小程序代码**，可能影响审核
